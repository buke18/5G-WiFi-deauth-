# 5G-WiFi-deauth-
2.4Ghz&amp;5Ghz双频WiFi deauth 安全测试模块（共分为两个版本：一个没有oled屏幕，需要手机端连接192.168.1.1）
# 项目一、不带OLED屏幕版本
可能不支持wpa3和Apple的蜂窝移动网络，不支持WiFi7路由器，其他的路由器应该都是可以使用这个工具测试的
